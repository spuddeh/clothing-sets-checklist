# Clothing Sets Checklist

**Clothing Sets Checklist** is a tool for collectors who want to complete specific clothing sets (e.g., Role Sets, Iconic Outfits).

## 🚀 Features

* **Set Grouping:** Organizes items by their outfit set (e.g., "Media Set", "Rockerboy Set").
* **Automation (v2.0):**
  * **Proximity Scanner:** Passively detects uncollected items nearby (50m) and marks them on your HUD.
  * **Smart Pause:** Suspend logic during menus/loading for zero overhead.
* **Smart Scanning:**
  * **Ownership Check:** Automatically scans your Inventory AND Wardrobe (Transmog) to detect owned items.
  * **Base ID Resolution:** Handles internal Game ID variations automatically.
* **Cheat Features:**
  * **Give Item:** A "Gift" button allows you to spawn the item directly into your inventory if you can't find it.
  * **Show IDs:** Option to display the internal `Item.ID` for debugging or console commands.

## 🎮 Usage

1. Open the CET Overlay.
2. The mod will automatically scan your inventory and wardrobe.
3. Owned items are marked with a checkmark.
4. Missing items show their location (if fixed) or description.
5. **Need it now?** Enable "Lazy Mode" to teleport or use the **[Gift]** button to spawn it.

## ⚙️ Settings

* **Show Base ID:** Toggles the display of the raw TweakDB ID for each item.
